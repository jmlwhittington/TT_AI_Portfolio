Machine Learning Model for Genre Prediction of Video Game Covers from Sixth-Generation Consoles

A Teachable Machine model trained on thirty sixth-generation console (PS2, original Xbox, GameCube, roughly equally distributed) game covers across three genres (platformer, roleplaying, and adventure; again, roughly equally distributed), input into an interface for testing new examples.

There are two iterations of this: the first merely features a confusion matrix that updates with each successive input; the second introduces a fourth category ("Low Confidence") for anything guessed under 75% on all categories, as well as a second confusion matrix for second-most guessed categories for tracking patterns.
